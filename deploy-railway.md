# Инструкция по деплою Telegram-бота на Railway (deploy-railway.md)

[Railway](https://railway.app/) — это одна из лучших и самых доступных платформ для развертывания Python-ботов и баз данных в cloud.

## 🛠 Пошаговый алгоритм развертывания

### Шаг 1: Подготовка репозитория
1. Убедитесь, что файлы `bot/main.py` и `bot/requirements.txt` находятся в вашем репозитории на GitHub.
2. В корне репозитория должен лежать файл `Procfile` (или Railway автоматически определит запуск по `requirements.txt`). Если нужен свой Procfile:
   ```text
   web: python bot/main.py
   ```

### Шаг 2: Создание проекта в Railway
1. Зайдите на [Railway.app](https://railway.app/) и авторизуйтесь через GitHub.
2. Нажмите кнопку **`+ New Project`** в правом верхнем углу.
3. Выберите **`Deploy from GitHub repo`**.
4. Найдите и выберите ваш репозиторий с проектом.

### Шаг 3: Настройка переменных окружения (Environment Variables)
1. После того как карточка сервиса появится на дашборде Railway, кликните по ней.
2. Перейдите во вкладку **`Variables`**.
3. Нажмите **`+ New Variable`** и добавьте ваши ключи из `.env`:
   - `TELEGRAM_BOT_TOKEN` = `1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ...`
   - `MASTER_TELEGRAM_ID` = `ваш_личный_telegram_id`
   - (Если подключаете PostgreSQL, добавьте `DATABASE_URL`).

### Шаг 4: Запуск и дебаг
1. Перейдите во вкладку **`Deployments`**. Вы увидите процесс сборки (Build).
2. Railway автоматически установит Python и зависимости из `requirements.txt`.
3. После успешного деплоя статус изменится на **`Active`** (зеленый индикатор).
4. Нажмите на вкладку **`Logs`**, чтобы убедиться, что в логах есть строчка:
   `🚀 Python Telegram Bot запущен!`
5. Откройте вашего бота в Telegram, нажмите `/start` и протестируйте подачу заявки. Всё работает 24/7!
